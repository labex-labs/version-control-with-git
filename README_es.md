Aprenda Control de Versiones con Git. Este módulo cubre la Configuración e Inicialización de Git, Inicialización de un Repositorio Local, Historial y Cambios, Conceptos Básicos de Ramificación (Branching), Ignorar Archivos con Gitignore. Dominará estas habilidades esenciales de Linux a través de laboratorios prácticos y desafíos del mundo real.
